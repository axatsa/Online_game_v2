ТЕХНИЧЕСКОЕ ЗАДАНИЕ: Интерфейс Учителя (Teacher Portal)

Проект: ClassPlay (SaaS)

1. Введение

Интерфейс учителя — это основная рабочая зона платформы. Это светлый, "воздушный" веб-интерфейс, адаптированный для использования на ноутбуках и интерактивных досках.
Цель: Экономия времени учителя на подготовку к урокам и геймификация процесса обучения.

2. Дизайн и Навигация

Стилистика:

Фон: Светлый (bg-slate-50).

Акцентный цвет: Бордовый (#991B1B).

Меню: Горизонтальное, липкое (Sticky Header).

Структура Меню:

Мой класс (Настройки профиля класса).

Генераторы (Создание материалов).

Инструменты (Рулетка, Таймер).

Библиотека Игр (Запуск интерактивных игр).

Профиль: Выпадающее меню с аватаром (инициалы) и кнопкой "Выход".

3. Функциональные Модули

3.1. Модуль "Мой Класс" (Killer Feature)

Назначение: Настройка контекста для AI. Учитель описывает свой класс один раз, и AI использует это для всех генераций.

Поля ввода:

Название класса: (например, 3 "Б").

Язык обучения: Переключатель RU / UZ.

Уровень: Слабый / Средний / Сильный.

Описание (Промпт): Текстовое поле, где учитель пишет: "Любят футбол, проходят тему дроби".

Логика: Данные сохраняются в состоянии приложения (или БД) и добавляются в системный промпт Gemini API.

3.2. Модуль "AI Генераторы"

Главная страница: Карточки доступных генераторов (Математика, Кроссворд) с иконками.

Интерфейс Генератора (Split View):

Слева (Настройки):

Выбор предмета, сложности, количества вопросов.

Кнопка "Сгенерировать" (вызов Gemini API).

Справа (Превью):

Визуализация листа А4.

Отображение сгенерированного контента (примеры или сетка).

Кнопки: Скачать (PDF), Печать, Поделиться.

Обработка ошибок: Если AI недоступен, выдавать заранее заготовленный шаблон (Fallback).

3.3. Модуль "Библиотека Игр"

Назначение: Запуск игр на проекторе/доске.

Список Игр:

Битва Знаний (Tug of War): Командная игра. Две кнопки на экране, перетягивание ползунка.

Мемори (Memory Matrix): Поиск пар.

Своя Игра (Jeopardy): Викторина по категориям.

Режим запуска: Игры открываются в полноэкранном режиме или в модальном окне, чтобы скрыть интерфейс браузера.

4. Требования к API

Endpoint: POST /generate-content (или прямой вызов Gemini API с фронтенда для MVP).

Формат ответа: Строгий JSON.

{
  "title": "Тема урока",
  "questions": ["Вопрос 1", "Вопрос 2"]
}
