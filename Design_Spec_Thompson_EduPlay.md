ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ТЗ) НА ДИЗАЙН UI/UX

Проект: Thompson EduPlay Platform

1. ОБЩАЯ КОНЦЕПЦИЯ И СТИЛИСТИКА (Visual Identity)

Дизайн должен транслировать атмосферу престижной международной школы: академичность, строгость, но с современным игровым подходом (Edutainment). Интерфейс должен быть "воздушным", с использованием большого количества белого пространства и мягких теней.

1.1. Цветовая палитра

Primary (Основной бренд): #991B1B (Deep Burgundy / Бордовый). Использовать для логотипа, активных кнопок (CTA), акцентных блоков и выделения уровней.

Secondary (Вторичный): #0F172A (Slate 900). Для заголовков, навигационных элементов и темных фонов.

Background (Фон): #F8FAFC (Slate 50). Светлый, почти белый, "чистый" фон страницы.

Cards (Карточки): #FFFFFF (White). Чистый белый с легкой тенью.

Accents (Акценты):

Золотой/Янтарный: Для рейтингов, звезд, монет (наград).

Серый (Muted): #64748B для второстепенного текста.

1.2. Типографика

Заголовки (Headings): Шрифт с засечками (Serif), например, Playfair Display, Merriweather или Lora. Придает статусность.

Основной текст (Body): Гротеск (Sans-serif), например, Inter, Roboto или Open Sans. Обеспечивает читаемость интерфейса.

1.3. Формы и Эффекты

Скругления: Border-radius: 16px (lg) для кнопок и 24px (2xl) для крупных карточек.

Тени: Мягкие, рассеянные тени (box-shadow), создающие эффект левитации при наведении.

Glassmorphism: Использование эффекта матового стекла (backdrop-blur) для липких шапок (sticky headers) и всплывающих окон.

2. СТРУКТУРА СТРАНИЦ (Сценарий Пользователя - Ученик)

2.1. Страница Авторизации / Регистрации (Login/Sign Up)

Компоновка: Split Screen (Разделенный экран).

Левая часть (50%): Фотография школы или кампуса в высоком качестве с наложением темно-бордового градиента. Крупный логотип школы и слоган по центру.

Правая часть (50%): Форма входа на белом фоне.

Элементы формы:

Поля: "Email / ID ученика", "Пароль". Стиль: минималистичные input с серой обводкой, при фокусе — бордовая.

Кнопка: Широкая, цвет #991B1B, скругленные углы. Текст: "Войти в класс".

Ссылка: "Забыли пароль?" и "Подать заявку (для родителей)".

2.2. Главная панель (Dashboard / Home) — Текущий дизайн

Шапка (Header):

Липкая (Sticky), фон белый полупрозрачный (Blur).

Логотип слева: Буква "T" в бордовом квадрате + Текст "THOMPSON".

Меню по центру: "Обучение", "Рейтинг", "Профиль" (активный пункт выделен цветом и подложкой).

Справа: Индикатор "Стрик" (огонек + дни подряд), Поиск, Аватар.

Блок "Герой" (Hero Section):

Крупная белая карточка с приветствием ("Рады видеть, Алекс").

Справа внутри карточки: Виджеты статистики (Текущий уровень, XP). Бордовый блок для уровня, Белый для XP.

Декор: Абстрактные размытые круги (бордовый и серый) на фоне для глубины.

Фильтры:

Горизонтальный скролл кнопок (Pills). Активная кнопка — темная (Slate 900), неактивные — белые с обводкой.

Сетка игр (Grid):

Карточки игр 3-4 в ряд.

При наведении: Картинка слегка увеличивается (Zoom), появляется кнопка "Играть".

На карточке: Лейбл категории (сверху слева), Рейтинг (звезда, снизу справа).

2.3. Страница "Игровой процесс" (Game Interface)

Режим: Focus Mode (Минимум отвлекающих факторов).

Верхняя панель:

Кнопка "Назад" (стрелка).

Название игры по центру.

Таймер и Текущий счет справа.

Игровая область:

Центральный контейнер (Canvas/Iframe) с соотношением 16:9. Темная подложка вокруг игры.

Скругленные углы у игрового окна.

Нижняя часть:

Инструкция к игре (Текст в 3 колонки: Цель, Управление, Навык).

2.4. Страница Рейтинга (Leaderboard)

Стиль: Таблица в виде списка карточек.

Топ-3: Выделены визуально.

1 место: Золотая обводка/иконка, увеличенный размер.

2 место: Серебро.

3 место: Бронза.

Список:

Строки: Аватар, Имя, Класс (бейджик), Количество XP (жирным шрифтом).

Чередование цвета фона строк (Zebra striping) — очень легкий серый и белый.

2.5. Профиль Ученика (Profile)

Шапка профиля: Крупный аватар, Имя, Класс, Статус (например, "Математический гений").

Блок "Достижения" (Badges): Сетка круглых иконок-наград. Полученные — цветные, неполученные — черно-белые (Grayscale).

Графики:

Круговая диаграмма успеваемости по предметам (Математика, Английский и т.д.).

История активности (список последних игр).

3. СТРУКТУРА СТРАНИЦ (Административная панель / Учительская)

Дизайн админки должен быть более утилитарным и плотным, чем у студентов, но сохранять цветовую гамму.

3.1. Админ-Дашборд

Сайдбар (Слева): Темный (#0F172A). Пункты: "Обзор", "Ученики", "Игры", "Статистика", "Настройки".

Сводка (Widgets):

4 карточки сверху: "Всего учеников", "Активных сегодня", "Сыграно игр", "Средний балл".

График активности: Линейный график посещаемости за неделю (Бордовая линия).

3.2. Управление Учениками (User Management)

Таблица данных:

Колонки: ID, Имя, Класс, Email, Последний вход, Действия (Редактировать/Блок).

Функции: Поиск, Фильтр по классу, Кнопка "Добавить ученика" (Primary color).

3.3. Управление Играми (CMS)

Список игр: Таблица с миниатюрами обложек.

Добавление игры (Модальное окно или отдельная страница):

Поля: Название, Категория (Select), Сложность, URL игры (iframe src), Загрузка обложки.

Тумблер: "Опубликовать / Скрыть".

4. АДАПТИВНОСТЬ (Mobile)

Навигация: На мобильных устройствах верхнее меню сворачивается в "Бургер" (справа). При открытии — выезжающая шторка сбоку или сверху на белом фоне.

Сетка игр: Превращается в 1 колонку (карточки друг под другом).

Таблицы: В админке и рейтинге таблицы получают горизонтальный скролл, либо трансформируются в карточки.

5. АССЕТЫ И ИКОНКИ

Иконки: Использовать набор Lucide React или Phosphor Icons. Стиль: Outline (контурные), толщина линий 1.5px или 2px.

Иллюстрации: Минималистичные 3D иконки или качественные фото (Unsplash) с наложением фирменного цвета.

6. UI KIT (Компоненты)

Кнопка Primary: Фон #991B1B, Текст белый, Hover: #7F1D1D.

Кнопка Secondary: Фон Прозрачный, Обводка #E2E8F0, Текст #0F172A.

Input: Фон #F8FAFC, Бордер #E2E8F0, Радиус 12px.

Badge (Бейдж): Пастельные тона (фон) + Насыщенный текст (например, светло-красный фон + красный текст).
